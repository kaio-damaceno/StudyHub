## ❓ Perguntas Frequentes

**P: Onde os meus dados são salvos?**  
R: Tudo é armazenado localmente no seu computador. Nenhum dado é enviado para servidores.

**P: Como faço backup?**  
R: Nas configurações, clique em "Backup" para salvar seus dados.

**P: Posso usar offline?**  
R: Sim! A maioria das funcionalidades trabalha sem internet.
