## 🎯 Roteiros de Aprendizado Recomendados

### Roteiro 1: "Sou Novo no SRS"

```
Semana 1:
  → [Primeiros Passos]
  → [Curva de Esquecimento](fundamentals/forgetting-curve.md)
  → [Tipos de Cards](fundamentals/card-types-guide.md)
  ✓ Criar primeiro deck

Semana 2:
  → [Workflow: ENEM](tutorials/workflows-reais.md#caso-1)
  → [Dicas de Performance](performance/optimization-tips.md)
  ✓ Completar 7 dias de revisões

Semana 3:
  → [Adapt-SRS Básico](advanced/adapt-srs-deep-dive.md#conceito-de-estabilidade)
  → [Progresso Tracking](performance/progress-tracking.md)
```

---

### Roteiro 2: "Migrando de Anki"

```
Dia 1:
  → [Study Hub vs. Anki](comparison/vs-anki.md)
  → [Import/Export](tutorials/import-export.md)
  ✓ Importar seu deck Anki

Dia 2-7:
  → [Adapt-SRS Overview](advanced/adapt-srs-deep-dive.md)
  → Revisar seus cards normalmente
  → Observar recomendações do sistema

Semana 2:
  → [Customize Adapt-SRS](tutorials/customizing-adapt-srs.md)
  → [Performance Tips](performance/optimization-tips.md)
  ✓ Otimizar workflow
```

---

### Roteiro 3: "Sou Desenvolvedor"

```
Dia 1:
  → [Arquitetura](advanced/architecture.md)
```
