# 🎯 Guia Rápido - Study Hub

## Começando

### Primeira Execução
1. Abra Study Hub
2. Você verá uma aba em branco com a barra de busca
3. Configure seu mecanismo de busca preferido nas configurações

### Navegação Básica

#### Barra Superior
- **Logo**: Clique para ir à página inicial
- **Barra de endereço**: Digite URLs ou termos para buscar
- **Seletor de busca**: Escolha Google, Bing ou DuckDuckGo

#### Sidebar Esquerda
Acesso rápido a todas as funcionalidades:
- 🌐 **Navegador** - Abas e histórico
- 🧠 **Flashcards** - Suas coleções de estudo
- 📝 **Notas** - Editor de anotações
- 🗺️ **Mapas Mentais** - Diagramas visuais
- 📚 **Documentos** - Leitor de PDFs
- ⚙️ **Configurações** - Personalize

---

## 🌐 Navegador

### Abas
- **Nova aba:** `Ctrl+T` ou clique em `+`
- **Fechar aba:** `Ctrl+W` ou clique em `X`
- **Alternar abas:** `Ctrl+Tab`

### Busca
- Digite na barra de endereço para buscar
- Cole URLs completas para navegar
- Mude o motor de busca nas configurações

### Favoritos
- Clique ⭐ para adicionar um sites
- Acesse na barra lateral
- Organize seus sites importantes

---

## 🧠 Flashcards

### Criar Coleção
1. Clique em "Flashcards"
2. Clique em "Nova Coleção"
3. Digite um nome
4. Confirme

### Estudar
1. Abra uma coleção
2. Clique em "Estudar"
3. Para cada cartão, classifique como:
   - **Fácil** - Revisar menos
   - **Normal** - Revisar regularmente
   - **Difícil** - Revisar com frequência

### Importar/Exportar
- **Importar:** Clique em "Importar" e selecione arquivo JSON/TXT
- **Exportar:** Clique em "Exportar" para salvar

---

## 📝 Notas

### Criar Nota
1. Clique em "Notas"
2. Clique em "Nova Nota"
3. Digite seu conteúdo

### Formatação
- **Negrito:** `Ctrl+B`
- **Itálico:** `Ctrl+I`
- **Sublinhado:** `Ctrl+U`
- **Código:** `` Ctrl+` ``

### Organização
- Crie pastas para categorizar
- Use tags para busca rápida
- Conecte blocos relacionados

---

## 🗺️ Mapas Mentais

### Criar Mapa
1. Clique em "Mapas Mentais"
2. Clique em "Novo Mapa"
3. Digite o tema central

### Adicionar Nós
- Clique em "Novo Nó"
- Digite o conteúdo
- Arraste para posicionar

### Conectar
- Clique em um nó e arraste para outro
- Uma linha conectará os conceitos

---

## 📚 Documentos

### Abrir Documento
1. Clique em "Documentos"
2. Clique em "Abrir Arquivo"
3. Selecione PDF, imagem ou Word

### Ler
- Use zoom: scroll do mouse
- Barra lateral: navegue por páginas
- Tela cheia: pressione `F11`

### Extrair Flashcards
1. Abra um PDF
2. Clique em "Extrair Flashcards"
3. Selecione seções
4. Crie coleção automaticamente

---

## ⚙️ Configurações

### Aparência
- **Tema:** Claro/Escuro
- **Tamanho:** Ajuste de fonte
- **Zoom:** Padrão para documentos

### Navegação
- **Mecanismo Padrão:** Google/Bing/DuckDuckGo
- **Página Inicial:** Configure seu início

### Dados
- **Backup:** Faça cópia de segurança
- **Restaurar:** Recupere dados
- **Limpar:** Cache e histórico

## 🔥 Dicas Rápidas

✅ Use `Ctrl+L` para focar na barra de endereço

✅ Revise regularmente cartões marcados como "Difícil"  
✅ Use mapas mentais para planejar ensaios  
✅ Faça backup de seus dados regularmente
