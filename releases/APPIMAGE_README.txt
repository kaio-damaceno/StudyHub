# 📦 Study Hub Linux - AppImage

Este arquivo será preenchido após compilação para Linux.

Para compilar a versão Linux:

```bash
npm run dist:linux
```

O arquivo gerado terá aproximadamente 180-200 MB.

**Nota:** A compilação requer ferramentas de build do Linux configuradas.

Consulte o [Guia de Compilação](./COMPILACAO.md) para mais detalhes.
