# 📥 Instaladores - Study Hub v1.3.3

## Download e Instalação

### Windows 10/11
**Arquivo:** `Study Hub Setup 1.3.3.exe`

**Como instalar:**
1. Baixe o arquivo `.exe`
2. Execute o instalador
3. Clique em "Próximo" e aceite os termos
4. Escolha pasta de instalação (padrão recomendado)
5. Conclua a instalação
6. Study Hub abrirá automaticamente

### Linux (AppImage)
**Arquivo:** `Study Hub 1.3.3.AppImage`

**Como instalar:**
```bash
chmod +x Study\ Hub\ 1.3.3.AppImage
./Study\ Hub\ 1.3.3.AppImage
```

### Linux (Debian/Ubuntu)
**Arquivo:** `study-hub-1.3.3.deb`

**Como instalar:**
```bash
sudo apt install ./study-hub-1.3.3.deb
```

Procure por "Study Hub" no menu de aplicações.

---

**Versão:** 1.3.3  
**Tamanho:** ~180 MB  
**Requisitos:** 2 GB RAM, 200 MB espaço em disco
