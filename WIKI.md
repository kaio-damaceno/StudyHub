## Introdução

Study Hub é uma plataforma completa de produtividade para estudantes. Ele combina navegação web, ferramentas de anotação, flashcards inteligentes e gerenciamento de documentos em uma única interface intuitiva.

**Versão Atual:** 1.3.3  
**Plataformas Suportadas:** Windows 10+, Linux (Ubuntu 18.04+)

---

## Instalação

### Requisitos do Sistema

**Windows:**
- Windows 10 ou superior
- 200 MB de espaço em disco
- 2 GB de RAM (recomendado 4 GB)

**Linux:**
- Ubuntu 18.04 ou equivalente
- 200 MB de espaço em disco
- 2 GB de RAM (recomendado 4 GB)

### Processo de Instalação

#### Windows
1. Baixe o arquivo `StudyHub-Setup.exe` da página de releases
2. Execute o instalador
3. Clique em "Próximo" e aceite os termos
4. Escolha a pasta de instalação (padrão recomendado)
5. Conclua a instalação
6. Study Hub abrirá automaticamente

#### Linux (AppImage)
1. Baixe o arquivo `study-hub-*.AppImage`
2. Abra o terminal no diretório de Downloads
3. Execute: `chmod +x study-hub-*.AppImage`
4. Execute: `./study-hub-*.AppImage`

#### Linux (Debian/Ubuntu)
1. Baixe o arquivo `study-hub-*.deb`
2. Execute: `sudo apt install ./study-hub-*.deb`
3. Execute Study Hub a partir do menu de aplicações

## Interface Principal

Study Hub possui três componentes principais:

### Barra Superior
- **Logo**: Clique para voltar à página inicial
- **Barra de Endereço**: Digite URLs ou termos de busca

### Sidebar Esquerda
- **Navegador**: Acesse abas e histórico

- **Configurações**: Personalizar sua experiência

### Área Principal
- Exibe o conteúdo da seção selecionada
- Responsiva e adaptável para diferentes tamanhos

## Navegador

### Recursos Básicos

#### Abas
- **Nova Aba**: Clique no ícone "+" ou pressione `Ctrl+T`
- **Fechar Aba**: Clique no "X" ou pressione `Ctrl+W`
- **Navegar**: Clique na aba desejada

#### Busca e Navegação
- Digite na barra de endereço para buscar
- Digite URLs completas para navegar
- Use `Ctrl+L` para focar na barra de endereço

#### Histórico
- Clique em "Histórico" na sidebar
- Sua atividade de navegação é salva automaticamente
- Limpe histórico nas configurações quando desejar

### Favoritos
- Clique no ícone ⭐ para adicionar favoritos
- Acesse favoritos no painel lateral
- Organize seus sites mais visitados

### Dicas
- Use `Ctrl+Tab` para alternar entre abas
- `Ctrl+Shift+T` reabre a última aba fechada
- Mecanismo de busca padrão pode ser alterado nas configurações

## Flashcards

### Criar uma Coleção

1. Acesse a seção **Flashcards**
2. Clique em "Nova Coleção"
3. Digite um nome para sua coleção
4. Clique em "Criar"

### Adicionar Cartões

1. Abra uma coleção existente
2. Clique em "Novo Cartão"
3. Adicione a pergunta (frente)
4. Adicione a resposta (verso)
5. Salve o cartão

### Sistema de Estudo

#### Modo de Estudo
1. Abra uma coleção
2. Clique em "Estudar"
3. Estude os cartões seguindo a sequência
4. Classifique cada resposta como: Fácil, Normal ou Difícil
5. O sistema ajusta automaticamente a frequência de repetição

#### Sistema de Repetição Espaçada
- **Cartões Fáceis**: Revisados menos frequentemente
- **Cartões Normais**: Revisados em intervalo médio
- **Cartões Difíceis**: Revisados com mais frequência

### Estatísticas
- Visualize seu progresso geral
- Veja quantos cartões estão em cada nível
- Acompanhe sua taxa de aprendizado

### Importar e Exportar

#### Importar
1. Clique em "Importar Coleção"
2. Selecione arquivo JSON ou TXT
3. Escolha se deseja mesclar com coleção existente

#### Exportar
1. Abra uma coleção
2. Clique em "Exportar"
3. Escolha formato (JSON ou TXT)
4. Selecione local para salvar

## Notas

### Editor de Notas

#### Criar Nova Nota
1. Clique em "Notas" na sidebar
2. Clique em "Nova Nota"
3. Digite seu título
4. Comece a escrever

#### Formatação de Texto
- **Negrito**: `Ctrl+B` ou **texto**
- **Itálico**: `Ctrl+I` ou *texto*
- **Sublinhado**: `Ctrl+U` ou <u>texto</u>
- **Código**: `` `código` `` ou ` Ctrl+` `

#### Blocos de Notas
- Crie blocos independentes de conteúdo
- Reorganize blocos com drag & drop
- Conecte blocos relacionados com linhas

#### Organização
- Organize notas em pastas
- Use tags para categorização
- Busque suas notas rapidamente

## FAQ

### Geral

**P: Posso usar Study Hub offline?**  
R: Sim, a maioria das funcionalidades funciona offline. Apenas a busca na web requer conexão.

**P: Onde meus dados são armazenados?**  
R: Todos os seus dados são armazenados localmente no seu computador no diretório de dados da aplicação.

**P: Posso sincronizar entre dispositivos?**  
R: Atualmente, não há sincronização automática. Você pode exportar dados e importá-los em outro dispositivo.

### Flashcards

**P: Como funciona o sistema de repetição espaçada?**  
R: Cartões marcados como "Fácil" aparecem com menos frequência. "Normal" em intervalo padrão, e "Difícil" com frequência aumentada.

**P: Posso importar de outras aplicações?**  
R: Sim, exporte suas coleções no formato JSON de outras aplicações e importe no Study Hub.

### Documentos

**P: Quais formatos são suportados?**  
R: PDF, JPG, PNG, WebP, e arquivos Word (.docx).

**P: Posso anotar documentos?**  
R: Sim, você pode fazer highlights e adicionar notas em anexo (anotações salvas separadamente).

### Performance

**P: O aplicativo está lento.**  
R: Tente limpar cache nas configurações. Se o problema persistir, reinicie a aplicação.

**P: Como faço backup de meus dados?**  
R: Vá em Configurações > Armazenamento > Backup para realizar backup manual.
