##  Funcionalidades

### 🌐 Navegador Completo
- Navegação com abas intuitiva
- Múltiplos mecanismos de busca (Google, Bing, DuckDuckGo)
- Compatibilidade com websites modernos
- Histórico e favoritos sincronizados

### 🧠 Sistema de Flashcards
- Criar e organizar suas coleções de estudo
- Sistema inteligente de repetição
- Acompanhe seu progresso com estatísticas
- Importar e exportar suas coleções facilmente

### 📝 Editor de Notas
- Editor rico com formatação completa
- Organize suas ideias em blocos conectáveis
- Criação de listas e estruturas complexas
- Salvo automaticamente

### 🗺️ Mapas Mentais
- Visualize suas ideias em diagrama visual
- Conecte conceitos relacionados
- Organize hierarquicamente seus pensamentos
- Exporte seus diagramas

### 🎯 Biblioteca de Documentos
- Abra e estude PDFs diretamente
- Suporte para imagens e documentos Word
- Recursos de leitura inteligente
- Extraia automaticamente flashcards

##  Instalação

### Windows
1. Acesse a página de [**Releases**](../../releases)
2. Baixe o arquivo `StudyHub-Setup.exe`
3. Execute o instalador e siga as instruções
4. Study Hub estará pronto para usar

### Linux
1. Acesse a página de [**Releases**](../../releases)
2. Baixe o arquivo `.AppImage` ou `.deb`
3. Para AppImage: Execute diretamente
4. Para .deb: `sudo apt install ./study-hub.deb`

---

##  Primeiros Passos

1. **Abra Study Hub** - Inicie com uma nova aba
2. **Configure seu navegador** - Escolha seu mecanismo de busca preferido
3. **Crie uma coleção de flashcards** - Organize seus estudos
4. **Adicione suas notas** - Comece a documentar seu aprendizado
